Installation from Sources
=========================

Installing the Agent
--------------------

Clone the project from Github: ::
    $ git clone https://github.com/eProsima/Micro-RTPS
    $ mkdir Micro-RTPS/build && cd Micro-RTPS/build

On Linux, execute: ::

    $ cmake -DTHIRDPARTY=ON ..
    $ make
    $ sudo make install
